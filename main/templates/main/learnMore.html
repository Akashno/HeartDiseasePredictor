Learn more about this project

This project analyzes the heart disease patient dataset with proper data processing. Then, different models are trained and predictions are made based on that models. Classifiers like Logistic regression, Naive Bayes, Decision trees, Random forest, and support vector machines are used as model algorithms for predicting the results.

There is a total of 6 Algorithms used Here for Model Training.
  • Logistic regression Algorithm
  • Gaussian Naive Bayes Algorithm
  • Decision tree Algorithm
  • Support vector machines Algorithm
  • KNN Algorithm

  • Python: Since python is the best fit in the field of data science and machine learning we chose python as our environment for this project.
  • Django: Django is used as the web support for this project. Django is so efficient in integrating with machine learning.
  • Sci kit learn Libraries: Classifiers packages are from the Sci kit learn library
